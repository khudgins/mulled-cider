# Mulled Cider for Alacritty

Terminal color scheme for the Alacritty GPU-accelerated terminal emulator.

## Installation

### Import (recommended)

Add to your `alacritty.toml`:

```toml
[general]
import = [
  "/path/to/alacritty-mulled-cider.toml"
]
```

### Copy

Alternatively, copy the `[colors]` sections from `alacritty-mulled-cider.toml` directly into your `alacritty.toml`.

### Config file locations

- **macOS**: `~/.config/alacritty/alacritty.toml`
- **Linux**: `~/.config/alacritty/alacritty.toml`
- **Windows**: `%APPDATA%\alacritty\alacritty.toml`

Restart Alacritty to apply.

## What's Themed

- Primary background and foreground
- Cursor colors
- Normal and bright ANSI colors (all 16)
- Selection colors
