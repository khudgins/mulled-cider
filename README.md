# Mulled Cider

A warm, autumnal color theme that brings the cozy feeling of autumn to your development environment.

## Flavors

| Application | File | Docs |
|---|---|---|
| VS Code | `themes/mulled-cider-color-theme.json` | [docs/vscode.md](docs/vscode.md) |
| Alacritty | `alacritty-mulled-cider.toml` | [docs/alacritty.md](docs/alacritty.md) |
| Neovim | `colors/mulled-cider.lua` | [docs/neovim.md](docs/neovim.md) |
| Obsidian | `obsidian-mulled-cider.css` | [docs/obsidian.md](docs/obsidian.md) |
| tmux | `tmux-mulled-cider.conf` | [docs/tmux.md](docs/tmux.md) |

| Role | Colors |

| Backgrounds | `#2B1F1A` `#241A16` `#1F1612` |
